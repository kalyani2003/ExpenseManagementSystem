# Expense Management System

A full-stack backend Expense Management System built using Spring Boot, MongoDB, Redis, Kafka, JWT Authentication, Docker, and Swagger.

---

## Features

- User Authentication using JWT
- CRUD Operations for Expenses
- MongoDB Database Integration
- Redis Caching
- Apache Kafka Integration
- Swagger API Documentation
- Pagination & Sorting
- Logging with SLF4J
- Global Exception Handling
- Docker Containerization
- JUnit & Mockito Testing

---

## Tech Stack

- Java 17
- Spring Boot 3
- Spring Security
- MongoDB
- Redis
- Apache Kafka
- Docker
- Swagger OpenAPI
- Gradle
- JUnit & Mockito
